# Student Performance Analysis

## Internship Project - Cognetix Technology

This project analyzes student academic performance using Python, Pandas, Matplotlib, and Seaborn.

---

## Features

- Data Validation
- Subject Average Analysis
- Top Student Identification
- Percentage Calculation
- Correlation Heatmap
- Histogram Visualization
- Performance Analysis

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

---

## Insights

- Reading and writing scores are strongly correlated.
- Student performance distribution is centered around average scores.
- Top students showed consistent performance across all subjects.
